# SQL-Data-Analysis-Portfolio
A collection of SQL data analysis projects demonstrating exploration, aggregation and analytical querying using real-world and synthetic datasets. Each dataset is broken into three digestible sections: beginner-level queries, intermediate analytical and aggregation scenarios, and advanced challenges using window functions and deeper SQL logic.

</br>

### Real World
- **Social Media Addiction & Relationships in Students** - This dataset focuses on students social media usage amongst students, focusing on daily usage hours, sleep duration and addiction score metrics. Enables exploration of behavioral trends across demographic segments such as age and gender. Well suited for practicing aggregation, cohort analysis and behavioral segmentation. Kaggle link <a href = "https://www.kaggle.com/datasets/adilshamim8/social-media-addiction-vs-relationships" target = "_blank">here</a>.
- **Accidental Drug Related Deaths in Connecticut** - This dataset contains records of accidental drug related deaths from 2012-2024, including demographic attributes (age, sex, race) and geographic indicators such as residence and injury location in addition to toxicology results for multiple substances. The data is fashioned for public health analytics, trend detection, co-occurance modeling and risk profiling at both demographic and regional level.  From the official catalog of Data.gov, link can be found here <a href = "https://catalog.data.gov/dataset/accidental-drug-related-deaths-2012-2018" target = "_blank">here</a>.
