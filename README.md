# SQL-Data-Analysis-Portfolio
A collection of SQL data analysis projects demonstrating exploration, aggregation and analytical querying using real-world and synthetic datasets. Each dataset is broken into three digestible sections: beginner queries, intermediate analytical and aggregation scenarios, and advanced challenges using window functions and deeper SQL logic.

## Featured Datasets
- AI Impact on Student Performance - Synthetic dataset found on Kaggle, link [here]([url](https://www.kaggle.com/datasets/ankushnarwade/ai-impact-on-student-performance)).
- Social Media Addiction & Relationships in Students - Real world dataset, Kaggle link [here]([url](https://www.kaggle.com/datasets/adilshamim8/social-media-addiction-vs-relationships)).
- Various US Medical Costs from 2010-2020 - Real world dataset, Kaggle link [here]([url](https://www.kaggle.com/datasets/waqi786/medical-costs)).
- Sleep Health and Lifestyle - Synthetic dataset, Kaggle link [here]([url](https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset)).
