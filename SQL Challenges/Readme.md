# Insurance Cost Analysis

## Dataset Summary
This project analyzes an insurance data containing demographic and health related attributes in addition to medical insurance charges.

### Columns
- age: Age of policyholder
- sex: male/female
- bmi: Body mass index, integer
- children: Number of dependents
- smoker: True or false
- region: Location, broken into 4 regions
- charges: Individual insurance charges

### Goal
To Use SQL and explore cost drivers of medical insurance. Create and identify high risk groups and utilize progressive querying 
techniques ranging from basic aggregation to more advanced analytical logic. 

### Challenge Structure
Each dataset is broken into 3 progressive sections:
- **Beginner**: Basic filtering, grouping, and summary statistics
- **Intermediate**: Conditional logic, categorical bucketing, and multi-column aggregations
- **Advanced**: Window functions, risk scoring logic, and multi-step analytical reasoning
